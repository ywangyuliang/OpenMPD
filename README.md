## Repository branches

| Branch | Description |
|---|---|
| `main` | Main branch of the repository |
| `release-final` | **Final version** delivered to the supervisor and the public — complete refactor and improvement of the project |
| `version-after-fix-base-translator` | Version of the base translator after reviewing and fixing previous errors *(created during the TFG)* |
| `yuliang-wang-task` | Development of task support *(created during the TFG)* |
| `yuliang-wang-task-profiling` | Testing and profiling results of task support *(created during the TFG)* |
| `caijie-wu-halo` | Development related to halo *(created during the TFG)* |

> **For the latest version and documentation, see the [`release-final`](https://github.com/ywangyuliang/OpenMPD/tree/release-final) branch.**
